# 🏠 House Price Prediction – End-to-End Machine Learning Project (Week 1)

## 🎯 Project Overview
This repository contains an end-to-end Machine Learning regression pipeline built to estimate the selling price of houses based on their structural characteristics and amenities. This project serves as a practical implementation of fundamental Data Science concepts including SQL analytics, Exploratory Data Analysis (EDA), Data Cleaning, Feature Engineering, and Multiple Linear Regression.

---

## 🎯 Business Problem
A real estate firm requires an automated tool to predict house prices based on various input features. Providing accurate price estimations helps both buyers and sellers make informed decisions.

* **Input Features:** House characteristics (Area, Bedrooms, Bathrooms, Stories, Parking, Air Conditioning, etc.)
* **Target Output:** Predicted Price (in ₹)
